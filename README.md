# Intro

This is the repository for CultureHub's website.
The main frontend code can be found in `client/`.
The setup and config for Sanity Studio can be found in `sanity/`.

# Development

## Prerequisites

* Git
* NodeJS
* pnpm (`npm install -g pnpm`)

## Getting started

If you're making changes to the schema or content editor refer to the [README in studio/](/studio/README.md).

If you're making changes to the frontend code refer to the [README in client/](/client/README.md).
